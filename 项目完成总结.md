## 🛠️ 技术栈

### 后端
- **框架**: FastAPI 0.104.1
- **数据库**: SQLite 3（可切换PostgreSQL 16）

- **认证**: JWT + bcrypt
- **邮件**: QQ SMTP
- **AI**: DeepSeek API + 自定义Agent系统

## 📝 配置说明

### 必需配置

**.env文件**：
```bash
# 应用配置
APP_NAME=印记
DEBUG=true

# 数据库
DATABASE_URL=sqlite+aiosqlite:///./yinji.db

# JWT
SECRET_KEY=your-secret-key-change-in-production

# SMTP邮箱（已配置）
SMTP_HOST=smtpdm-ap-southeast-1.aliyuncs.com
SMTP_EMAIL=noreply@yingjiapp.com
QQ_EMAIL_AUTH_CODE=lkvxwqnfjwvpdief

# DeepSeek API（需要配置）
DEEPSEEK_API_KEY=your-deepseek-api-key
DEEPSEEK_BASE_URL=https://api.deepseek.com/v1
```

### API Key获取

**DeepSeek API**：
1. 访问：https://platform.deepseek.com/
2. 注册账号
3. 获取API Key
4. 配置到`.env`文件
