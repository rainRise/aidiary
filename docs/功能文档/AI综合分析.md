# AI 综合分析

> 让 AI 帮你回顾一段时间的日记，给出深度洞察。

---

## 模块定位

AI 综合分析是一个**按需触发**的功能，用户主动点击「开始分析」后，后端会读取指定时间窗口内的日记，通过 RAG 检索提取关键片段，再调用大模型进行深度分析，返回结构化的洞察报告。

与成长中心的区别：成长中心是**被动展示**（打开就能看），综合分析是**主动请求**（需要等待生成）。

路由：`/analysis`（需登录）

---

## 使用流程

1. 用户选择分析的**时间窗口**（默认 90 天）
2. 点击「开始分析」
3. 等待后端调用大模型（通常需要 10~30 秒）
4. 页面展示分析报告

---

## 后端分析流程（综合分析）

前端调用 `POST /api/v1/ai/comprehensive-analysis`，执行以下步骤：

```
1. 从数据库读取时间窗口内的所有日记（最多 120 篇）
2. 用 RAG 服务将日记切片 → 生成摘要块 + 原文块
3. 按 5 个维度分别检索相关片段：
   ├── 情绪趋势（焦虑、平静、开心、低落...）
   ├── 连续事件（重复出现的困扰/问题）
   ├── 关键转折（决定、变化、契机）
   ├── 成长线索（进步、反思、改变）
   └── 关系主题（家人、同事、朋友...）
4. 去重后保留最多 18 条证据片段
5. 拼成 prompt，单次调用大模型生成 JSON 报告
```

### RAG 检索细节

使用 `rag_service.py` 实现的本地轻量 RAG，不依赖外部向量数据库：

- **切片**：日记按句子边界分段（最大 260 字，40 字重叠）
- **检索**：BM25 关键词匹配 + 多维加权排序
- **排序公式**：`55% 文本相关性 + 14% 时间新近度 + 9% 重要性 + 8% 情绪强度 + 8% 主题重复度 + 4% 人物命中 + 2% 来源加成`
- **去重**：Jaccard 相似度 ≥ 72% 的片段去重，每篇日记最多取 2 条

## 多智能体工作流（单篇分析）

后端还有一条**多智能体流水线**（`POST /api/v1/ai/analyze`），用于单篇日记的深度分析。由编排器 `AgentOrchestrator` 协调 4 个智能体依次工作：

```
Step 0 → 上下文收集智能体：汇总用户画像 + 时间轴上下文
Step A → 时间线管家：从日记中提取事件并结构化
Step B → 萨提亚分析师（4 个子步骤）：
  ├── B1: 情绪层分析
  ├── B2: 认知与信念层分析
  ├── B3: 存在层分析
  └── B4: 疗愈回复生成
Step C → 社交内容生成器：基于分析结果输出多版本文案
```

每个智能体独立调用大模型，结果逐步传递给下一个。分析结果包含：
- 时间轴事件（结构化摘要、情绪标签、重要性评分）
- 萨提亚五层冰山分析（行为 → 情绪 → 认知 → 信念 → 存在）
- 疗愈性回复
- 社交文案建议

> 目前综合分析页面 `/analysis` 走的是 RAG + 单次大模型路径，多智能体流水线暂未接入该页面。

## 记忆与数据积累

| 使用场景 | 是否使用历史日记 | 记忆方式 |
|----------|----------------|---------|
| 综合分析 `/analysis` | ✅ 读取时间窗口内的全部日记 | 本地 RAG（BM25 检索） |
| 映记精灵对话 | ✅ 检索相关日记片段 | Qdrant 向量检索（如已配置）或本地 RAG 兜底 |
| 单篇分析 | ✅ 接收时间轴上下文 | 编排器传入 |

> Qdrant 需要在 `.env` 中配置 `QDRANT_URL` 和 `QDRANT_API_KEY` 才会启用，未配置时所有检索回退到本地 BM25 RAG。

---

## 分析报告包含什么

| 板块 | 内容 |
|------|------|
| 综合总结 | 100~180 字概括这段时间的整体状态 |
| 关键主题 | AI 提炼出的 3 个核心主题 |
| 情绪趋势 | 情绪走向的变化描述 |
| 连续信号 | 反复出现的困扰或模式 |
| 关键转折点 | AI 识别出的情绪拐点事件 |
| 成长建议 | 基于以上分析给出的温和建议 |

---

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 时间窗口 | 90 天 | 分析最近多少天的日记 |
| 最大日记数 | 120 篇 | 避免 token 超限，超过时取最近的 |

---

## 交互细节

- 分析期间显示加载动画，无法重复触发
- 分析失败时显示错误提示，可重试
- 报告生成后停留在页面上，刷新页面后需重新分析（不缓存）
- 左上角返回按钮 → 回到首页
