# 成长中心

> 用日记画出你的情绪地图，看见自己的变化。

---

## 模块定位

成长中心是映记的**核心分析页面**，把你过去一段时间的日记浓缩成可视化的图表。  
不需要自己翻日记——打开成长中心，一眼看到这段日子过得怎么样。

路由：`/growth`（旧路径 `/timeline` 会自动重定向过来）

---

## 页面结构

从上到下依次排列：

```
页头：← 返回  |  成长中心  |  写日记 →
├── 心情日历
├── 能量值趋势图
├── 萨提亚关键词卡片
└── 人际关系图谱（占位，后续迭代）
└── 当日详情
```

背景色为米白 `#FAF9F7`，整体风格与首页保持一致。

## 心情日历

### 长什么样

按月展示，每天是一个**同心圆环**，7 列排列（周日 ~ 周六）。

每个圆环包含三层信息：

| 看什么 | 表达什么 | 数据来源 |
|--------|---------|---------|
| **外环颜色** | 当天的主要情绪 | 日记的 `emotion_tag` |
| **内圆明暗** | 能量高低（亮 = 能量足，暗 = 能量低） | `energy` 值 1~10 |
| **环的粗细** | 当天写了多少事（粗 = 事件多） | 事件密度 `density` |

「环的粗细」的计算规则——事件优先，日记兜底：

| 场景 | 粗细值 | 说明 |
|------|--------|------|
| 写了 1 篇日记，AI 已分析 | 1 | 1 篇日记对应 1 条事件 |
| 写了 1 篇日记，AI 提炼出 3 个事件 | 3 | 按事件条数 |
| 写了 3 篇日记，都还没分析 | 3 | 没有事件时用日记篇数 |
| 当天什么都没写 | 0 | 显示空心圆环 |

没有日记的日子显示为一个淡灰色幽灵圆，今天用虚线外环标识。

日历底部有一行小图例，解释颜色/明暗/粗细/今天标记的含义。

### 情绪色谱

每种情绪有独立颜色，覆盖 60+ 种情绪标签，按色温分组：

| 色系 | 代表情绪 |
|------|---------|
| 暖阳系（琥珀/暖橘） | 开心、快乐、兴奋、幸福 |
| 柔绿系 | 成就感、成长、自豪、充实 |
| 暖棕系 | 感恩、感动、温暖 |
| 薄荷系 | 平静、放松、释然、满足 |
| 薰衣草系 | 期待、好奇 |
| 雾蓝系 | 疲惫、无聊 |
| 灰紫系 | 困惑、纠结、迷茫 |
| 暗橘系 | 担忧、焦虑、紧张、压力 |
| 青灰系 | 失落、难过、悲伤、孤独 |
| 赤陶系 | 愤怒、崩溃、恐惧 |
| 中性灰 | 平淡、一般 |

> 色谱定义在 `EmotionChart.tsx` 的 `EMOTION_COLORS` 中，全站共用，包含模糊匹配兜底。

### 月份切换

- 左右箭头切换上一月 / 下一月
- 默认显示当月

### 交互

**悬停**：
- 弹出浮窗，内容由后端 AI 生成：
  > *"3月22日 · 平静"*
  > *"这天你去爬山了，感觉很舒畅。"*
- 浮窗数据按需加载（hover 时请求），拿到后缓存

**点击**：
- 如果当天有日记 → 跳转日记详情
- 如果只有事件无日记 → 在页面底部的「当日详情」卡片中展示

## 能量值趋势图

### 长什么样

一条**平滑曲线面积图**，X 轴为日期，Y 轴为能量值（1~10 分）。

曲线下方有渐变色填充：高能量区偏琥珀金，低能量区偏藏青蓝。

### 时间范围

提供 **7 天 / 30 天 / 90 天** 三个 Tab 切换，默认 30 天。

### 关键事件标注

当某天的能量值相比前一天**变化 ≥ 2 分**时，在曲线节点上方显示一个小图标：

- 💼 工作相关
- 🤝 人际关系
- 🌙 健康 / 作息
- � 成就
- ✨ 其他

点击图标 → 弹出小卡片，显示事件摘要 + 「查看日记」链接。

### 交互

点击曲线上的任意节点 → 页面底部「当日详情」卡片更新为该天的信息。

## 萨提亚关键词卡片

### 设计思路

不用传统词云（太杂乱），而是按照**萨提亚冰山模型**的 5 个层级，分别展示你日记中的高频关键词。

### 布局

横向排列 5 张卡片，移动端变为纵向：

| 卡片 | 标题 | 底色 |
|------|------|------|
| 行为层 | 我做了什么 | 暖橙 |
| 情绪层 | 我感受到 | 淡紫 |
| 认知层 | 我在想 | 薄荷绿 |
| 信念层 | 我相信 | 浅蓝 |
| 渴望层 | 我渴望 | 浅粉 |

每张卡片内显示 5~8 个关键词，字越大表示出现越频繁。

### 数据来源

后端从日记文本中提取关键词，按预定义的词表匹配归入对应层级。时间范围与能量趋势图一致。

## 当日详情

页面底部有一个「当日详情」卡片，展示：

- 日期
- 能量值、愉悦值、事件数
- 当天所有事件的摘要列表，每条可点击跳转到对应日记

通过点击日历圆环或趋势图节点来切换查看不同日期。

---

## 数据自动更新

保存日记时**不做任何分析**，成长中心的数据统一由后端**每日定时任务**（午夜 0:00）生成：

1. 扫描昨天和今天所有有日记记录的用户
2. 对尚未分析的日记，先用规则提取基础事件，再调用大模型精炼摘要、情绪标签、重要性评分
3. 结果写入 `TimelineEvent` 表，下次打开成长中心即可看到

> 唯一的例外：心情日历中 hover 某天时，会按需调用大模型生成当日洞察（`GrowthDailyInsight`），生成后缓存到数据库，不重复请求。

---

## 交互联动

- 心情日历点击某天 → 跳转日记详情
- 趋势图点击事件标注 → 弹出事件摘要 → 可跳转日记
- 趋势图点击节点 → 底部当日详情更新
- 映记精灵在此页面可感知展示的数据，支持提问："我最近状态怎么样？"

---

## 边界情况

| 情况 | 处理 |
|------|------|
| 日记少于 3 篇 | 显示引导："再写几篇日记，成长洞察将逐渐清晰" |
| 当月没有日记 | 日历全部显示淡灰占位圆 |
| 能量值没有大波动 | 趋势图不显示事件标注，只展示平滑曲线 |
