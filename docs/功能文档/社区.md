# 社区功能

## 功能概述

用户在社区里可以发布自己的生活日常，支持文字、图片、视频等多种形式。
可以对他人的动态进行点赞、评论、收藏等操作。
每个人都有自己的收藏夹，也有浏览记录（只有点开一个具体的动态才会记录浏览记录，而不是浏览列表时记录）。
用户在社区里可以选择匿名发表想法。

## 匿名机制

匿名帖子不允许再次编辑（保护发布者，也减少恶意修改）。
匿名用户的互动（点赞/评论）也匿名。

## 核心定位

不做开放式广场，做"情绪共鸣圈"。
开放式社区容易变成刷屏、焦虑放大器，心理健康类 app 尤应避免。

建议按"情绪场景"组织社区，而不是时间流：

社区分为6个圈子，用户可以根据自己的情况来在合适的圈子发帖

- [焦虑] 我们都在焦虑的事
- [失落] 那些低落的时刻
- [成长] 突破自我的瞬间
- [平静] 享受当下的时光
- [困惑] 想不明白的时候

不做粉丝数/关心数体系。
