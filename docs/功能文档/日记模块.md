# 日记模块

> 映记的核心——写日记、看日记、管理日记。

---

## 模块定位

日记模块是映记最基础也最重要的功能，所有的 AI 分析、成长中心、社区内容都建立在日记数据之上。

---

## 路由

| 页面 | 路由 | 说明 |
|------|------|------|
| 日记列表 | `/diaries` | 查看所有日记，支持搜索和过滤 |
| 日记详情 | `/diaries/:id` | 查看单篇日记完整内容 |
| 新建日记 | `/diaries/new` | 写一篇新日记 |
| 编辑日记 | `/diaries/:id/edit` | 修改已有日记 |

---

## 日记包含哪些信息

| 字段 | 说明 |
|------|------|
| 标题 | 可选，不填时列表显示内容前几个字 |
| 内容 | 正文，必填 |
| 日记日期 | 默认今天，可以改成其他日期（补写昨天的日记） |
| 情绪标签 | 多选，如「开心」「焦虑」「平静」等 |
| 重要性评分 | 1~10 分，默认 5，影响成长中心的能量值 |
| 图片 | 支持上传多张图片 |

---

## 日记列表

- 分页展示，每页 20 条
- 支持按**日期范围**和**情绪标签**过滤
- 每条日记显示标题/内容摘要、日期、情绪标签

---

## 写日记 / 编辑日记

- 富文本编辑器
- 支持拖拽或点击上传图片
- 保存时只写入 Diary 表，不触发任何 AI 分析（分析集中在每日 0 点定时任务处理）

---

## 删除日记

- 支持删除单篇日记
- 删除后相关的时间轴事件也会清除

---

## 图片上传

- 上传到后端 `uploads/diary_images/` 目录
- 文件名自动生成 UUID，防止冲突
- 返回可访问的 URL 路径
