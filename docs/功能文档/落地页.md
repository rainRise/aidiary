# 落地页

> 用户打开映记看到的第一个页面，负责传达产品价值、引导注册。

---

## 模块定位

落地页是未登录用户的**默认入口**。已登录用户访问时会自动跳转到首页仪表盘。

路由：`/welcome`

---

## 页面结构

从上到下分为几个区块：

- **导航栏**：Logo + 产品名「印记」+ Beta 标签 + 功能/定价锚点链接 + 登录/注册按钮
- **主视觉区**：大标题 + 副标题 + 立即开始按钮
- **功能亮点**：用卡片展示核心功能（AI 情绪分析、成长轨迹、隐私保护等）
- **定价说明**：展示免费/付费方案
- **页脚**：隐私政策、服务条款、退款政策链接

整体视觉风格：米白暖色渐变背景，与应用内页面保持一致。

---

## 交互

- 点击「立即开始」或「免费注册」→ 跳转 `/register`
- 点击「登录」→ 跳转 `/login`
- 导航栏功能/定价链接 → 页内锚点滚动
- 页脚法律链接 → 跳转对应页面（`/privacy`、`/terms`、`/refund`）

---

## 法律政策页面

落地页关联三个独立页面，无需登录即可访问：

| 页面 | 路由 |
|------|------|
| 隐私政策 | `/privacy` |
| 服务条款 | `/terms` |
| 退款政策 | `/refund` |
