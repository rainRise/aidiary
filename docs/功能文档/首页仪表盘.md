# 首页仪表盘

> 登录后看到的第一个页面，一眼掌握自己的日记状态。

---

## 模块定位

仪表盘是映记的**主页**，聚合展示用户的日记统计、情绪分布和各模块的快捷入口。

路由：`/`（需登录）

---

## 页面结构

### 顶部导航

- 左侧：Logo + 产品名
- 右侧：用户头像，点击展开下拉菜单（个人设置、退出登录）

### 统计卡片区

三张统计卡片横向排列：

| 卡片 | 内容 |
|------|------|
| 日记总数 | 用户写过的所有日记数量 |
| 本月日记 | 当月已写日记数量 |
| 最常情绪 | 出现次数最多的情绪标签 |

### 情绪气泡图

用气泡图展示用户所有日记中的情绪分布：
- 每种情绪是一个气泡，大小对应出现次数
- 气泡颜色使用全站统一的情绪色谱
- 悬停显示具体情绪名称和次数

### 快捷入口

卡片式布局，引导用户前往各个功能模块：

| 入口 | 跳转 | 图标 |
|------|------|------|
| 写日记 | `/diaries/new` | 钢笔 |
| 我的日记 | `/diaries` | 书本 |
| 成长中心 | `/growth` | 嫩芽 |
| 综合分析 | `/analysis` | 闪光 |
| 个人设置 | `/settings` | 设置 |
| 社区 | `/community` | 消息气泡 |

### 最近日记

展示最近几篇日记的标题和日期，点击可跳转日记详情。

---

## 数据来源

- 统计数据：前端从日记列表（最多 50 条）中本地计算
- 情绪气泡图：从日记的 `emotion_tags` 字段聚合
